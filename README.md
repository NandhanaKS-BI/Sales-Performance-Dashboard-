# 📊 Sales Performance Dashboard (Excel)

## 📌 Project Overview
This project presents an interactive Sales Performance Dashboard developed using Microsoft Excel. 
The dashboard analyzes sales and profit performance across different regions and categories using dynamic Pivot Tables and visualizations.

---

## 🎯 Objective
The objective of this dashboard is to:
- Monitor overall sales performance
- Track total profit
- Analyze regional and category-wise sales
- Support data-driven decision making

---

## 🛠 Tools & Techniques Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
- Data Cleaning
- Excel Formulas (SUM, IF, etc.)

---

## 📈 Key Metrics Included
- Total Sales
- Total Profit
- Sales by Region
- Sales by Category
- Monthly Sales Trend

---

## 📊 Dashboard Features
- Interactive slicers for filtering data
- Dynamic Pivot Tables and charts
- KPI summary cards for quick insights
- Region-wise and category-wise comparison
- Clean and structured dashboard layout
